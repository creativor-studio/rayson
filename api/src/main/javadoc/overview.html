Rayson is An API server and it's development framework.

Main concepts.

  • Protocol {@link org.rayson.api.Protocol}, API definition.
  • RSON {@link org.rayson.rson}, built in structured data for serializing.
  • API server implementation in rayson.server library.
  • Rayson java client in rayson.client library.
  • Filter {@link org.rayson.api.Filter} and filter chain {@link org.rayson.api.FilterChain}.
